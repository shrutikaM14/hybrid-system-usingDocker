# Shrutika Hybrid System – Docker Command Reference

## Project Folder

```bash
cd ~/shrutika
```

---

# Docker Compose Commands

## Start All Containers

```bash
sudo docker compose up
```

---

## Start In Background (Recommended)

```bash
sudo docker compose up -d
```

---

## Rebuild Images + Start

```bash
sudo docker compose up --build
```

---

## Stop Running Attached Containers

```text
CTRL + C
```

---

## Stop and Remove Containers

```bash
sudo docker compose down
```

---

# Container Management

## Show Running Containers

# Common Problems

## localhost:8000 not opening

Check:

```bash
sudo docker ps
```

If fastapi-container stopped:

```bash
sudo docker logs fastapi-container
```

Then restart:

```bash
sudo docker restart fastapi-container
```

---

## System Freezing

Stop Neo4j:

```bash
sudo docker stop neo4j-container
```

Neo4j uses high RAM.
